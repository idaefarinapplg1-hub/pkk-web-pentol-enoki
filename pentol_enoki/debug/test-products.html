3. Manual Testing Checklist

Home Page

  • Home page terbuka tanpa error
  • Produk featured ditampilkan (maksimal 6)
  • Gambar produk muncul atau placeholder
  • Harga ditampilkan dengan format Rupiah
  • Tombol "Tambah ke Keranjang" berfungsi
  • Animasi saat menambahkan ke cart
  • Cart badge bertambah
  • Notifikasi sukses muncul

Menu Page

  • Menu page terbuka tanpa error
  • Semua produk ditampilkan
  • Filter kategori berfungsi
  • Smooth scrolling berfungsi
  • Custom scrollbar terlihat
  • Hover effect pada product card
  • Add to cart berfungsi

Admin - Tambah Produk

  • Login sebagai admin berhasil
  • Tab "Produk" dapat dibuka
  • Tombol "Tambah Produk" membuka modal
  • Form tampil dengan baik
  • Dropdown kategori terisi
  • Semua field dapat diisi
  • Validasi required field berfungsi
  • Submit form berhasil
  • Produk muncul di daftar
  • Notifikasi sukses muncul

Admin - Edit Produk

  • Tombol "Edit" membuka modal
  • Data produk ter-load di form
  • Semua field terisi dengan benar
  • Dapat mengubah data
  • Submit update berhasil
  • Perubahan tersimpan

Admin - Arsipkan Produk

  • Tombol "Arsipkan" menampilkan konfirmasi
  • Produk berhasil diarsipkan
  • Badge "Archived" muncul
  • Produk tidak muncul di home/menu
  • Tombol berubah jadi "Aktifkan"
  • Dapat mengaktifkan kembali

Admin - Hapus Produk

  • Tombol "Hapus" menampilkan warning
  • Produk berhasil dihapus
  • Produk hilang dari daftar
  • Produk tidak muncul di home/menu

Scenario 1: Produk Baru

1. Login sebagai admin

2. Tambah produk baru

3. Periksa di home page

4. Periksa di menu page

5. Test add to cart

6. Database Checks

Jalankan query berikut di phpMyAdmin:

Lihat Semua Produk:

SELECT * FROM products ORDER BY created_at DESC;

Lihat Produk Aktif:

SELECT * FROM products 
WHERE is_available = 1 AND is_archived = 0;

Lihat Produk dengan Kategori:

SELECT p.*, c.category_name 
FROM products p 
LEFT JOIN categories c ON p.category_id = c.category_id;
