2. Checklist Testing

  • Menu page terbuka tanpa error
  • Produk ditampilkan dengan benar
  • Gambar produk muncul (atau placeholder)
  • Filter kategori berfungsi
  • Tombol "Tambah ke Keranjang" berfungsi
  • Cart badge menampilkan jumlah item
  • Halaman bisa di-scroll ke bawah
  • Smooth scrolling berfungsi
  • Hover effect pada product card berfungsi
  • Responsive di mobile view

3. Browser Console Check

Buka Developer Tools (F12) dan periksa:

  • Console Tab: Tidak ada error merah
  • Network Tab: Request ke products.php status 200
  • Console Logs: Harus muncul:
    • Loading products...
    • Products result: {success: true, data: [...]}
    • Filtered products: X

4. Test Scroll Behavior

  1. Buka menu page
  2. Scroll ke bawah menggunakan mouse wheel
  3. Perhatikan smooth scrolling effect
  4. Periksa custom scrollbar (merah)
  5. Test di berbagai ukuran layar

5. Test Add to Cart

  1. Klik tombol "Tambah ke Keranjang" pada produk
  2. Perhatikan animasi scale pada product card
  3. Periksa notifikasi sukses muncul
  4. Periksa cart badge bertambah
  5. Buka halaman cart untuk verifikasi

6. Test Responsive

  1. Buka Developer Tools (F12)
  2. Toggle Device Toolbar (Ctrl+Shift+M)
  3. Test ukuran:
    • Mobile: 375px
    • Tablet: 768px
    • Desktop: 1200px
  4. Periksa layout menyesuaikan dengan baik

7. Troubleshooting

Jika menu tidak muncul:

  1. Periksa database memiliki data produk
  2. Test API langsung: products.php
  3. Gunakan debug page untuk melihat response
  4. Periksa console browser untuk error
  5. Clear browser cache (Ctrl+Shift+Delete)

Jika scroll tidak smooth:

  1. Update browser ke versi terbaru
  2. Clear cache dan reload (Ctrl+F5)
  3. Periksa CSS file ter-load dengan benar
